# 🔐 Gnosis VaultBot — Telegram Signer

This is a simple Telegram bot for interacting with your Safe.

## ✅ Features
- `/start` — welcome message
- `/deploy` — triggers Vercel deployment via webhook
- `/sign` — fetches pending Safe transactions
